# 🍅 FocusFlow - Pomodoro Timer für Android

**Meistern Sie Ihre Produktivität mit der eleganten Pomodoro-Technik!**

FocusFlow ist eine moderne, intuitive Pomodoro-Timer-App für Android mit striktem Focus auf Benutzerfreundlichkeit und Minimalismus.

---

## ✨ Features

### 🎯 Kernfunktionalität
- **Smart Pomodoro Timer** - 25 Min Arbeit / 5 Min Pause (konfigurierbar)
- **Auto-Start Sessions** - Timer startet automatisch nach Pausen!
- **Nahtlose Session Übergänge** - Einfach ☕ oder 🎯 drücken, um sofort zu wechseln
- **Pausie & Reset** - Volle Kontrolle über deine Sessions

### 📊 Tracking & Statistiken
- **Fokus-Minuten Tracking** - Sehe wie viel Zeit du heute fokussiert hast
- **Session Counter** - Zähle completete Pomodoro Sessions
- **Tägliche Statistiken** - Stunden-Anzeige auf einen Blick

### 🎨 User Experience
- **Magische Farbanimationen** - Blau (Arbeit) ↔ Grün (Pause)
- **Intuitive Steuerung** - Große, zugängliche Buttons
- **Klare Zeitanzeige** - MM:SS Format
- **Moderne Material 3 Design** - Jetpack Compose Framework

### 🔔 Benachrichtigungen
- **Session Completion Alerts** - Vibration + Notification wenn deine Arbeit/Pause endet
- **Haptisches Feedback** - Unterschiedliche Vibrationen für verschiedene Events

### ⚙️ Anpassbarkeit
- **Flexible Einstellungen** - Passe Arbeitsdauer (1-60 Min) und Pausendauer (1-30 Min) an
- **Settings im laufenden Betrieb** - Änderungen nur wenn Timer pausiert
- **Saubere Benutzeroberfläche** - Keine ablenkenden Elemente

---

## 🚀 Schnellstart

1. **App öffnen**
2. **▶️ drücken** - Timer startet 25 Minuten Fokus-Arbeit
3. **Arbeiten** - Konzentriere dich voll auf deine Aufgabe
4. **Automatische Pause** - Nach 25 Min startet die 5-Min Pause automatisch
5. **Wiederholen** - Einfach weitermachen!

**Tipp:** Nutze ☕ und 🎯 Buttons zum manuellen Wechsel auf Arbeit/Pause!

---

## 🎯 Die Pomodoro-Technik

FocusFlow implementiert die **Pomodoro-Technik** - eine bewährte Zeitmanagement-Methode:

1. **Wähle eine Aufgabe**
2. **Starte einen 25-Min Pomodoro** (intensiver Fokus)
3. **Arbeite konzentriert** - keine Ablenkungen!
4. **Mache eine 5-Min Pause** (Wasser trinken, Dehnen, etc.)
5. **Wiederhole** - Nach 4 Pomodoros, mache eine längere 15-30 Min Pause

**Studien zeigen:** Die Pomodoro-Technik erhöht die Produktivität um bis zu 50%!
